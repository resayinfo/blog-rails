Blog-Ruby-On-Rails
==========


A Rails 4.1 blog with devise, rolify, cancan, rspec, search, bootstrap css + js, iHover css,  jQuery, google fonts, and a gitignore.

$ cd blog-rails

$ rake db:migrate

$ rake db:seed

$ bundle exec rspec (tests unfinished)

$ rails s

log in with seeds, or create user account
